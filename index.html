PhyloProfile

Introduction

Phylogenetic profiles

  • presence/absence pattern of genes across a set of taxa
  • similar profiles <=> relative functions
  • additional information betwwen orthologs (e.g. sequence similarity, taxonomy distance, protein architecture similarity, GO similarity, etc.) -> reliable functional predictions

Introduction

PhyloProfile tool

  • Shiny(R)-based
  • integrate, dynamically visualize and explore multi-layered phylogenetic profiles
  • available online & standalone

  • dynamically filter & analyze profiles with different thresholds
  • dynamically change the resolution of the profile analysis
  • dynamically change profile properties

    • Multi-layers

    =

    presence/absence pattern + two additional information


    • sequence similarity
    • domain architectures similarity
    • gene Ontology-term-based semantic similarities
    • taxanomy distances
    • etc.
